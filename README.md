# Coderhouse: Desafio con Express
## Formas de probar el proyecto
### Iniciar con Handlebars:
```
npm run start-hbs
```
### Iniciar con Pug:
```
npm run start-pug
```
### Iniciar con EJS:
```
npm run start-ejs
```

## Definiciones
| Version de artefacto | Puerto |
| ----------- | ----------- |
| Handlebars | 8080 |
| Pug | 8081 |
| Ejs | 8082 |
